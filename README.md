**Лабораторная работа №5**

Структура проекта

**banking** - основная библиотека
**Transaction** - класс для работы с транзакциями
**Account** - класс для работы с банковскими счетами
**tests** - модульные тесты
**Сборка проекта**

```
mkdir build
cd build
cmake ..
make
```

**Тесты**

```
cd build
ctest
```
